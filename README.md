# GLAB 385.4 – MySQL Integration

## Files Included

### 1. connect.py
Demonstrates connecting to MySQL using mysql.connector.

### 2. create_table_app.py
Creates a `tasks` table inside the classicmodels database.

### 3. param_query_app.py
Inserts a record into the `tasks` table using parameterized queries.

## Requirements
- MySQL installed
- classicmodels database installed
- mysql-connector-python installed
